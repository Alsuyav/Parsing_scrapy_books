Проект Parsing_scrapy_books для парсинга сайта https://books.toscrape.com/
Этот проект представляет собой веб-скрейпер, написанный с использованием фреймворка Scrapy, для извлечения информации о книгах с сайта https://books.toscrape.com/.

Установка
1. Установите Python, если у вас его еще нет.
2. Установите необходимые зависимости, выполнив команду:
pip install -r requirements.txt

Запуск
Выполните команду для запуска скрейпера:

Копировать
scrapy crawl pages_crawl -o books_pages_all.json
Данная команда запустит скрейпер, который извлечет информацию о книгах и сохранит ее в формате JSON в файле books_pages_all.json.

Структура проекта
- scrapy.cfg - конфигурационный файл Scrapy.
- books/ - директория с исходным кодом скрейпера:
- spiders/ - директория с описанием веб-пауков (скраперов) проекта.
- pages_crawl.py - код паука для извлечения информации о книгах.
- requirements.txt - файл с необходимыми зависимостями проекта.

Настройка
При необходимости вы можете настроить параметры скрейпера в соответствии с вашими потребностями, изменив файл books_spider.py.
